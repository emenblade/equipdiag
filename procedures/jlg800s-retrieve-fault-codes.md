# Retrieve Fault Codes Using the Analyzer

**Equipment:** JLG 800S / 860SJ

1. Connect the hand held analyzer to the controller module.
2. Power up the control system (key switch on, emergency stop buttons pulled out).
3. Press ENTER at the HELP: PRESS ENTER display.
4. If a fault is present, the analyzer display will scroll the fault across the screen.
5. Press ENTER again to view the LOGGED HELP screen.
6. Use the RIGHT and LEFT arrow keys to scroll through the fault logs to view the last 25 faults.

> **Note:** The display will show HELP: EVERYTHING OK if no fault is detected.

> **Note:** POWER CYCLE (0/0) indicates a power up event with no associated fault count.
