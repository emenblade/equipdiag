# Restore the System Default Settings

**Equipment:** Genie S-60 / S-65 / S-60 TRAX / S-65 TRAX

1. Turn the key switch to ground control and pull out the red Emergency Stop button to the on position.  Result: The alarm should sound a long tone.
2. Activate and hold the calibration toggle switch for 5 seconds.  Result: The alarm should sound a tone for  1 second.
3. Push in the red Emergency Stop button to the off position.
