# Hydrostatic Coupler Installation

**Equipment:** JLG G5-18A

1. Clean the drive plate ( 4 ) and surrounding area.
2. Hydrostatic Coupler Installation
3. Install housing ( 2 ) to the engine with previously removed twelve bolts and washers ( 1 ). Torque bolts to 29 - 33 lb-ft (39 - 45 Nm).  NOTICE  Do Not use flushing compounds for cleaning purposes.  MAE33850  1 2 3
4. Transmission 6-6
