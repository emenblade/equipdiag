# Retrieve Ford Engine Fault Codes

**Equipment:** Genie S-60 / S-65 / S-60 TRAX / S-65 TRAX

> **Note:** Perform this procedure with the key switch in the off position.

> **Note:** Before the fault codes are displayed, the check engine light will blink a code 1-6-5-4 three times.

> **Note:** If any fault codes are present, the ECM will blink a three digit code three times for each code stored in memory.

> **Note:** Once a fault code has been retrieved and the repair has been completed, the ECM memory must be reset to clear the fault code from the ECM.

1. Open the ground controls side cover and locate the run/test toggle switch on the side of the ground control box.
2. Pull out the red Emergency Stop button to the on position at both the ground and platform controls.
3. Move and hold the run/test toggle switch to the test position.
4. Continue to hold the run/test toggle switch in the test position and count the blinks.  Note: Before the fault codes are displayed, the check engine light will blink a code 1-6-5-4 three times. After the fault codes, the check engine light will blink a code 1-6-5-4 three times again indicating the end of the stored codes.  Note: If any fault codes are present, the ECM will blink a three digit code three times for each code stored in memory. It will blink the first digit of a three digit code, pause, blink the second digit, pause, and then blink the third digit.
