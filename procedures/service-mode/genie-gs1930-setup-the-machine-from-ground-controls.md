# Setup the Machine from Ground Controls

**Equipment:** Genie GS-1930 / GS-2632 / GS-3232

> **Note:** Select a test area that is firm, level and free of obstructions.

> **Note:** Before updating the machine software, open a web browser and navigate to the following location for the Genie Machine Software Download portal, http://firmware.

> **Note:** There are two procedures available to update the machine software.

> **Note:** Some models will require opening the ground control box to access the GCON.

> **Note:** Verify it is the correct flash file.

> **Note:** Do not turn off power while the ECM is being reprogrammed.

> **Note:** Machines equipped with AGM batteries will need to select the AGM battery type using the GCON set parameters screen.

> **Note:** The display wil only show the IP address for 3 seconds.

> **Note:** The username and password are case sensitive.

1. Turn the key switch to ground controls.
2. Press and hold the ground control scroll up and scroll down buttons.  Ground Control Menu Buttons  1   scroll down button  2   enter button  3   scroll up button  4   escape button
3. Pull out the red Emergency Stop button to the on position at the ground controls.
4. Use the ground control menu buttons to select machine Model, Options and Speed Setup parameters. Follow the menu structure indicated on the ground control LCD display.
5. LCD display
6. Connect one end of a CAT5 Ethernet cable into the diagnostic port. Connect the other end into a PC or laptop.
7. Read and record the IP address.
8. Select the   Software   tab at the header bar.
9. Select the   Choose File   button and navigate to the downloaded Genie Flash file (.gff).  Note: Verify it is the correct flash file.
10. Select the   Load Flash File   button to verify the file is a .gff file.
11. Select the   Reprogram Device (s) button to start the ECM software update.
12. Select the   Run Application   button to exit the software update mode.
13. Push in the red Emergency Stop button and disconnect the CAT5 cable from the GCON.
14. Secure the diagnostic port cover using the retaining fasteners removed in step 5. Do not over tighten.  Note: Machines equipped with AGM batteries will need to select the AGM battery type using the GCON set parameters screen. Refer to the  Machine Setup   procedure in the Service and Repair manual that is appropriate for your machine.  Tip over hazard. Updating the SmartLink software may have impacted the machines default drive speeds. Tipping over the machine will result in death or serious injury.  Perform drive speed test. Refer to the   Maintenance Manual   that is appropriate for your machine.
15. Perform a function test. Refer to the Operator's Manual on your machine.
16. Return the machine to service.  Machine Application Mode
