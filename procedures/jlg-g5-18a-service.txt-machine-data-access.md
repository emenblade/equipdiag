# Machine Data Access

**Equipment:** JLG G5-18A

1. Machine Data Access  Machine and engine data can be accessed by connecting to the appropriate locations shown below. The machine analyzer plugs into the X159 connector ( 1 ) of the cab harness located behind the access panel on the right side of the dash. For machine diagnostics, refer to Section 9.17, “Diagnostic Trouble Codes (DTC) on Ground Module”. The engine data can be accessed through X46 connector ( 2 ) in the engine compartment utilizing engine diagnostic tool. For engine diagnostics, refer to Section 9.18, “Engine Diagnostic Trouble Code”.  
