# PRESSURE SETTING PROCEDURE

**Equipment:** JLG 800S / 860SJ

> ⚠ **STAND BY PRESSURE OR LOAD SENSE PRESSURE**
>
1. All applicable steps in Section 5.4, Start Up Proce- dures must be followed.
2. Set up of the function pump.
3. Adjustments made at the main valve block.
4. Adjustments made at the platform valve.  Set Up of the Function Pump  STAND BY PRESSURE OR LOAD SENSE PRESSURE

> **Note:** When replacing the level up or level down cartridge, the function should be cycled, and then the crack pressure value calibrated.
