# Install a Valve Cartridge

**Equipment:** Genie 1932 / 2032 / 2046

1. Dip the cartridge in clean oil to lube the O-rings.
2. Screw the cartridge in by hand until the top O-ring is met, then torque to specifications.
3. If required, install the valve coil(s) onto the valve stem. Install the coil nut onto the valve stem and torque to specifications.
