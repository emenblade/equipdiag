# Engine Installation

**Equipment:** JLG G5-18A

1. Position engine ( 3 ) in engine compartment being sure to line up the four motor mount holes. Replace bolts and torque to 185 lb-ft (251 Nm).
2. Reattach any previously labeled wire connections on the engine and transmission.
3. Install the hydraulic and fuel filters, and radiator/oil cooler assembly bracket assembly ( 2 ).
4. Install the previously removed rear fender ( 1 ).
5. Remove the plugs and connect all previously labeled fuel lines.
