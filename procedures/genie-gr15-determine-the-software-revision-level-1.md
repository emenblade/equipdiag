# Determine the Software Revision Level

**Equipment:** Genie GR-15 / GR-20

1. Remove the platform controls from the platform.
2. Place the platform controls close to the diagnostic display on the power unit side of the machine.  a c b d  a   joystick controller JC1 b   lift function select button BN9 c   red Emergency Stop button P2 d   yellow arrow
3. Turn the key switch to platform control and pull out the red Emergency Stop button to the on position at both the ground and platform controls.
4. Press the lift function select button.
5. Slowly move the joystick in the direction indicated by the yellow arrow.
