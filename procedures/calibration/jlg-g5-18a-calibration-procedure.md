# Calibration Procedure

**Equipment:** JLG G5-18A

1. With ignition key in OFF position, press and hold SYSTEM CHECK button ( 6 ) on LSI display and turn ignition key to engine START position. Release the ignition key when engine start is achieved, but continue to hold SYSTEM CHECK button on LSI display until the orange LED ( 4 ) on LSI display goes out and buzzer sounds (approximately 3 seconds). Release SYSTEM CHECK button ( 6 ).
2. The LEDs will perform a sequence. When only the third green LED ( 3 ) illuminates, press the SYSTEM CHECK button ( 6 ).
3. The first green LED ( 3 ) then illuminates. With no attachment and boom retracted, lift boom fully.
4. Press the SYSTEM CHECK button ( 6 ) on the LSI display and release. The first 3 green LEDs ( 3 ) will illuminate. The third then second green LEDs will go out as the calibration point is recorded.
5. The first green LED ( 3 ) goes out and buzzer sounds then the red LED illuminates ( 5 ).
6. Lower boom and without moving the machine, pick up the proper test weight ( W ) listed in the table.  MZ8640  6 5 4
