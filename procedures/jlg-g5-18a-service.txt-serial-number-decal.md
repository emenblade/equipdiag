# Serial Number Decal

**Equipment:** JLG G5-18A

1. Park machine on a firm, level surface, level machine, fully retract boom, lower boom, place transmission in (N) NEUTRAL, engage park brake and shut engine OFF.
2. Place a Do Not Operate Tag on both ignition key switch and steering wheel.
3. Open engine covers. Allow system fluids to cool.
4. Install dash cover.
5. Carefully pry center button ( 1 ) out of steering wheel.
6. Mark steering wheel and shaft to ensure proper installation. Remove nut ( 2 ) securing steering wheel ( 3 ) to splined steering column shaft.
7. Use a steering wheel puller to remove steering wheel ( 3 ) from splined shaft.  b.   Steering Wheel Installation
8. Label, disconnect and cap hydraulic hoses and fittings on steering orbital valve ( 7 ).
