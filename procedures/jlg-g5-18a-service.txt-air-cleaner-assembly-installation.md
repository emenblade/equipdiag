# Air Cleaner Assembly Installation

**Equipment:** JLG G5-18A

1. Install the air cleaner assembly ( 2 ) to the air cleaner mounting plate with the previously used hardware ( 4 ). Hand torque mounting bolts to 10 - 12 lb-ft (13.5 - 16 Nm).
2. Air Cleaner Assembly Installation
3. Adjust and tighten clamp before starting the machine.
4. Install air intake adapter duct ( 1 ) to air cleaner assembly ( 2 ) and tighten the clamp bolt.
6. Close and secure the engine cover.
7. Remove the Do Not Operate Tags from both the ignition key and the steering wheel.  CAUTION NEVER   run the engine with only the inner safety element installed.  MAE3040  3 4 2
