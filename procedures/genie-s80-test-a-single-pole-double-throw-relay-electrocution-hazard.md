# Test a Single Pole Double Throw Relay Electrocution hazard

**Equipment:** Genie S-80 / S-85

> **Electrocution hazard.**
>
1. Tag and disconnect all the wiring from the relay to be tested.
2. Tag, disconnect and plug the hydraulic hoses from the function pump. Cap the fittings on the pump.
3. Remove the pump mounting fasteners. Carefully remove the pump.
