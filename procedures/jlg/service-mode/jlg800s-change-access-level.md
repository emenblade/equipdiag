# Change the Access Level

**Equipment:** JLG 800S / 860SJ

> **Note:** When the analyzer is first connected, you will be in access level 2 which enables you to only view most settings. Level 1 access is required to change settings.

> **Note:** If the correct access level is not displayed, repeat the above steps.

1. Connect the hand held analyzer and power up the control system.
2. Scroll to the ACCESS LEVEL menu using the arrow keys.
3. Press ENTER to select the ACCESS LEVEL menu.
4. Using the UP or DOWN arrow keys, enter the first digit of the password (3).
5. Using the RIGHT arrow key, position the cursor to the right one space.
6. Use the UP or DOWN arrow key to enter the second digit of the password (3).
7. Continue using the arrow keys until all remaining digits of the password are shown: 33271.
8. Press ENTER.
