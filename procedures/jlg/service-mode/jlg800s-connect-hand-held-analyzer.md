# Connect the Hand Held Analyzer

**Equipment:** JLG 800S / 860SJ

> **Note:** The cable has a four pin connector at each end; the cable cannot be connected backwards.

> **Note:** If a fault is present after pressing ENTER at the HELP: PRESS ENTER display, the analyzer will scroll the fault across the screen. If no fault was detected, the display will read HELP: EVERYTHING OK. If powered up at the ground station, the display will read GROUND OK.

1. Connect the four pin end of the cable supplied with the analyzer to the controller module located in the platform box or at the controller module in the ground control box.
2. Connect the remaining end of the cable to the analyzer.
3. Power up the control system by turning the lower key switch to the platform or ground position and pulling both emergency stop buttons on.
4. Use the RIGHT and LEFT arrow keys to move between top level menu items.
5. Press ENTER to select a displayed menu item.
6. Press ESC to cancel a selected menu item.
