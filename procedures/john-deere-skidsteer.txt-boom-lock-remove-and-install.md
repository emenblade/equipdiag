# Boom Lock Remove and Install

**Equipment:** John Deere 326D / 328D / 329D / 332D / 333D

1. Park machine on level surface.
2. Lower boom to ground.
3. Raise operator's station. See Raising Operator's Station. (Operator's Manual.)
4. Remove cap screws (2).
5. Remove boom lock assembly from machine frame.
6. Repair or replace parts as necessary. See Boom Lock Disassemble and Assemble. (Group 3140.)
7. Install boom lock assembly to machine frame.
