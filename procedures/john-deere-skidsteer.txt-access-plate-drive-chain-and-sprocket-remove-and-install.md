# Access Plate Drive Chain and Sprocket Remove and Install

**Equipment:** John Deere 326D / 328D / 329D / 332D / 333D

1. Remove front attachment and park machine on flat level surface.
2. Raise and block machine. See Raising and Blocking Machine. (Group 1740.)
3. Drain chain case oil. See Change Chain Case Oil. (Operator's Manual.)
4. Remove chain case access plate. See Chain Case Access Plate Remove and Install. (Group 0250.)
