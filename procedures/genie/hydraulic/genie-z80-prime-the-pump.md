# Prime the Pump

**Equipment:** Genie Z-80 / Z-80/60

> **Note:** When replacing the hydraulic oil, it is recommended that the hydraulic tank be cleaned using a mild solvent and all hydraulic filters and strainers be replaced.

> **Note:** Always use pipe thread sealant when installing the suction hose fittings and the drain plug.

> **Note:** After the oil has been replaced, continue the scheduled quarterly maintenance inspection.

> **Note:** There are four types of hydraulic filters: tank return filter, medium pressure filter, high pressure filter and drive motor case drain filter.

> **Note:** Perform this procedure with the engine off.

> **Note:** The return filter may be mounted in the top of the hydraulic tank (internally) or outside of the tank (externally).

> **Note:** The medium pressure filter is for the charge pump and the high pressure filter is for all machine functions except the drive circuit and oscillating axle circuit.

> **Note:** The case drain filter is for the charge pump.

> **Note:** TRAX Models : Due to the additional components of the track system, which increase rotational friction, the track may not be able to be rotated manually.

1. Locate the hydraulic return filter housing on top of the hydraulic tank.
2. Remove the cap from the filter housing.
3. Lift the handle on the filter element and rotate the element counterclockwise to release the element from the housing.
4. Clean up and properly dispose of any oil that may have spilled.  Note: When replacing the hydraulic oil, it is recommended that the hydraulic tank be cleaned using a mild solvent and all hydraulic filters and strainers be replaced.  Note: Always use pipe thread sealant when installing the suction hose fittings and the drain plug.  Note: After the oil has been replaced, continue the scheduled quarterly maintenance inspection.
5. Install the new filter element into the filter housing.
6. Push the filter element down to be sure the O-ring on the element is fully seated into the housing.
7. Rotate the filter element clockwise to lock it in place.
8. Install the filter housing cap.
9. Use a permanent ink marker to write the date and number of hours from the hour meter on the oil filter housing.
