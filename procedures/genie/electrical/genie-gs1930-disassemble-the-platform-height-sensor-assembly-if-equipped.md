# Disassemble the Platform Height Sensor Assembly (if equipped)

**Equipment:** Genie GS-1930 / GS-2632 / GS-3232

> **Electrocution/burn hazard.**
>
1. Raise the platform 7 to 8 feet / 2.1 to 2.4 m.
2. Lift the safety arm, move it to the center of the scissor arm and rotate down to a vertical position.
3. Lower the platform onto the safety arm.  Crushing hazard. Keep hands clear of the safety arm when lowering the platform.
4. Turn the key switch to the off position.
5. Disconnect the battery pack from the machine.
6. Remove the fasteners securing the large platform height sensor cover to the large platform height sensor bracket.
7. Remove the platform height sensor cover.  Steer End  1   fasteners (platform height sensor cover)  2   platform height sensor cover
8. Tag and disconnect the platform height sensor from the platform height sensor harness.
9. Remove the fastener securing the platform height sensor assembly to the number 1 inner arm.
10. Remove the platform height sensor assembly from the number 1 pivot pin.
11. Remove the set screw securing the lever arm to the platform height sensor.
12. Remove the lever arm from the platform height sensor assembly.

> **Note:** When a Platform Height Sensor Assembly is removed or replaced, the platform overload system must be calibrated.
