# Fan Remove and Install

**Equipment:** John Deere 326D / 328D / 329D / 332D / 333D

1. Remove front attachment and park machine on flat level surface.
2. Raise boom and engage boom lock.
3. Raise engine cover and remove side panels. See Side Panel Remove and Install. (Group 1910.)
4. Remove fan guard.
5. Remove cap screws (1) and remove fan (2).
6. Repair or replace as necessary.
7. Install fan and cap screws.
8. Install fan guard and engine side panels. See Side Panel Remove and Install (Group 1910.). Lower engine cover.
