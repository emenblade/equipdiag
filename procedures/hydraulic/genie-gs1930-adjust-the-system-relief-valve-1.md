# Adjust the System Relief Valve

**Equipment:** Genie GS-1930 / GS-2632 / GS-3232

1. Locate the system relief valve on the function manifold (item 2).
2. Connect a 0 to 5000 psi / 0 to 350 bar pressure gauge to the test port on the function manifold (item 1).
3. Chock both sides of the wheels at the steer end of the machine.
4. Remove the platform controls from the platform.
5. Turn the key switch to platform controls and pull out the red Emergency Stop button to the on position at both ground and platform controls.  GS-1330m  1   test port  2   system relief valve  3   steer relief valve  GS-1530/32 and GS-1930/32  1   test port  2   system relief valve
