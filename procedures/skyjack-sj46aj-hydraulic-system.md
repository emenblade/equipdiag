# Hydraulic System

**Equipment:** Skyjack SJ46AJ / SJIII Series

1. Broken or defective drive pump shaft or coupling.  -   Check pump shaft and coupling. Replace if defective.
2. Hydraulic System  4.2-1   All Controls Inoperative
3. Stuck or defective Inlet Compensator/ differential sensing valve DSV1.  -   Check o-rings and clean valve. Check operation of valve. Repair or replace valve as required.
4. System pump P2 defective.  -   Test pump with flow meter. Replace if defective.
5. Plugged or defective high pressure filter F2.  -   Check filter. Replace if plugged or defective.  4.2-3   No Main Boom Up
6. Defective lift cylinder C3.  -   Check seals on cylinder. Replace as necessary. Replace cylinder if defective.  4.2-4   No Main Boom Down
7. Stuck or defective counterbalance valve CB5 or CB6.  -   Clean valve. Check O-rings on valve. Repair or replace valve as required.
8. Defective riser cylinder C6 or C7.  -   Check seals on cylinder. Replace as necessary. Replace cylinder if defective.  4.2-6   No Riser Boom Down
