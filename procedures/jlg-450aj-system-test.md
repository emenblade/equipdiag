# SYSTEM TEST

**Equipment:** JLG 450AJ

## TEST

1. Position Platform/Ground select switch to Platform .
2. Plug analyzer into connector at base of platform control box.
3. Pull out Emergency Stop switch and start engine.
5. Use arrow button to reach SYSTEM TEST and press Enter. Analyzer prompts to activate system test. Press Enter to activate.


> **Note:** In platform mode, the footswitch must be closed.

> **Note:** Tower lift valves are not tested if TOWER LIFT=NO.

> **Note:** Left/right jib valves are not tested unless JIB   =   SIDESWING.
