# Extend/Retract Cylinder Installation

**Equipment:** JLG G5-18A

1. Carefully install the extend/retract cylinder into rear of boom using a suitable sling.
3. Install mounting pin ( 21 ), spacers and washers through frame access hole on chassis and first boom section bore and cylinder barrel. Secure with hardware ( 20 ) removed earlier.
4. Remove plugs in bottom of extend/retract cylinder and install previously removed fittings ( 19 ).
5. Uncap and connect tubes ( 18 ) to previously labeled fittings at extend/retract cylinder. Apply locking compound to fittings.
6. Extend/Retract Cylinder Installation
7. Install cover plate ( 16 ) and hose clamps at rear of boom assembly.
8. Reinstall any miscellaneous parts, such as mirror, if removed previously.
