# Complete Boom Installation

**Equipment:** JLG G5-18A

1. Park the machine on a hard, level surface, place the transmission in (N) NEUTRAL, engage the park brake and shut the engine OFF.
2. Place a Do Not Operate Tag on both the ignition key switch and the steering wheel.
4. Balance boom assembly using suitable slings and carefully guide the boom into position for attachment to frame.
5. Align frame pivot bore with the boom assembly pivot bore. Install boom pivot pin ( 7 ).
6. Install capscrew ( 5 ). Apply locking compound and torque locknut ( 6 ) to 66 lb-ft (90 Nm).
