# Nut Hand Control Linkage Remove and Install

**Equipment:** John Deere 326D / 328D / 329D / 332D / 333D

1. Park machine on level surface.
2. Raise operator's station.
3. Remove footwell. See Footwell Remove and Install. (Group 1910.)
4. Loosen cable adjusting nuts (2) on cable bracket at control valve.
5. Remove lock nuts (1) from end of hydraulic control cables (4) at bellcranks (3).
