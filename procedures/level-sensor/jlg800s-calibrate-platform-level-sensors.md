# Calibrate the Platform Level Sensors

**Equipment:** JLG 800S / 860SJ

> **Note:** Perform this procedure for platform software prior to version P3.4.

1. Put the machine into ground mode.
2. Start the machine and plug in the analyzer.
3. Manually level the platform with the switch on the MTB.
4. Go to the ACCESS LEVEL 2 screen.
5. Enter the password 33271 to get into access level 1 mode.
6. Go to the CALIBRATIONS menu and press ENTER.
7. Use the RIGHT arrow key to go to the PLAT LEVELING screen.
8. Press ENTER. A "Calibrate?" prompt should appear.
9. Press ENTER again to calibrate the level sensors.
10. When calibration has been successful, "Cal. Complete" should appear.
11. Cycle power to the machine.
