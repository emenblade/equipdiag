GM   DIAGNOSTIC   TROUBLE   CODES  Malfunction   Indicator   Lamp   (MIL)  The   MIL   should   illuminate   when   the   key   is   in   the   on   position   and   the   engine   is   not   running.   This   feature   will  verify   that   the   lamp   is   in   proper   working   order.   If   the   lamp   does   not   illuminate   with   the   vehicle   key   on/engine   off,   repair  it   as   soon   as   possible.   Once   the   engine   is   in   start   or   run   mode,   the   MIL   should   turn   off.   If   the   lamp   remains   on   while   the  engine   is   in   the   start   or   run   mode   a   diagnostic   trouble   code   may   be   set.  Diagnostic   Trouble   Codes   are   set   when   the   ECM   (Electronic   Control   Module)   runs   a   diagnostic   self   test   and   the  test   fails.   When   a   DTC   is   set,   the   ECM   will   illuminate   the   MIL   and   also   save   the   DTC   in   memory.  Diagnostic   Trouble   Codes   (DTC)  The   Diagnostic   Trouble   Codes   can   be   read   by   using   either   the   MIL   lamp   or   a   laptop   computer   with   the   correct  interface   hardware   and   software.   If   more   than   one   DTC   is   detected,   start   the   diagnostic   repair   with   the   lowest   DTC  number   set   as   they   are   numbered   in   order   of   importance.  Example:   Having   DTC   112   and   DTC122   both   concerning   the   oxygen   sensor,   it   is   possible   that   by   repairing   DTC   112   first,  the   problem   causing   the   DTC   122   may   also   be   corrected.  Diagnostic   Trouble   Codes   can   be   cleared   from   memory   with   a   laptop   computer,   or   by   turning   the   ignition   key   to  the   OFF   position   and   removing   the   ECM   power   fuse   (F8)   for   15   seconds.  Using   MIL   to   flash   codes  With   the   switch   held   in   diagnostic   mode   (3.0L)   or   pins   1   and   4   jumped   on   the   diagnostic   connector   (1.6L),   turn  the   ignition   key   to   the   on   position.   The   system   will   now   enter   the   self   diagnostic   blink   code   mode.   Be   ready   with   pen   and  paper   to   write   down   any   codes   that   may   be   stored.  The   ECM   will   flash   the   MIL   indicator   with   a   pause   between   represented   numbers   that   represent   DTC   codes.   The  sequence   starts   with   code   1654.   Code   1654   confirms   the   system   has   entered   the   blink   code   mode.   The   ECM   will   flash  code   1654   (3)   times   before   displaying   the   actual   DTC   code   that   may   be   set.  Example:   One   short   blink   (pause)   six   short   blinks   (pause)   five   short   blinks   (pause)   four   short   blinks.  If   no   DTC   codes   are   found,   the   ECM   will   continue   to   flash   1654   only.   This   means   no   stored   DTC   codes   were   found.  The   diagnostic   connector   is   located   on   the   top   of   the   engine
