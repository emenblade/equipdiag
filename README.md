# equipdiag

Heavy equipment machine diagnostics reference.

## Structure

```
manufacturers/   - Per-brand docs, guides, and notes
systems/         - System-level info (engine, hydraulics, electrical, etc.)
fault-codes/     - Fault code references and cross-reference tables
procedures/      - Diagnostic procedures and workflows
specs/           - Machine specs, fluid capacities, torque values
schematics/      - Wiring diagrams, hydraulic schematics
tools/           - Tool setups, software configs, adapter info
```
